🔒 개인정보처리방침

최종 업데이트: 2024년 1월 19일

1. 수집하는 개인정보

네이버 부동산 분석 도우미 확장 프로그램은 다음과 같은 정보를 수집합니다:

  • 사용자가 네이버 부동산에서 조회하는 매물 정보 (가격, 면적 등)
  • 사용자가 직접 입력하는 분석 데이터

수집하지 않는 정보:

  • 개인 식별 정보 (이름, 이메일, 주소 등)
  • 네이버 로그인 정보
  • 검색 기록

2. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 부동산 가격 분석 및 시각화 제공
  • 면적별 가격 비교 기능 제공
  • JSON 형식 데이터 내보내기 기능

3. 정보 저장

  • 모든 데이터는 사용자의 브라우저에만 저장됩니다 (localStorage)
  • 서버로 전송되거나 외부에 공유되지 않습니다
  • 사용자가 직접 삭제할 수 있습니다

4. 제3자 공유

수집된 정보는 제3자에게 공유되지 않습니다.

5. 보안

데이터는 사용자의 로컬 브라우저에만 저장되며, 별도의 서버 전송이 없어 외부 유출 위험이 없습니다.

6. 사용자 권리

사용자는 언제든지:

  • 브라우저의 개발자 도구에서 localStorage를 확인할 수 있습니다
  • 저장된 데이터를 삭제할 수 있습니다
  • 확장 프로그램을 삭제하면 모든 데이터가 삭제됩니다

7. 문의
