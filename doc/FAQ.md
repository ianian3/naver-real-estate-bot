# 자주 묻는 질문 (FAQ)

## 📌 서비스 관련

### Q1. 이 서비스는 무엇인가요?
네이버 부동산 데이터를 수집하여 아파트 시세를 분석하고, 투자금(갭)을 자동으로 계산해주는 서비스입니다. 관심 있는 아파트를 등록하면 가격 변동 시 이메일로 알림을 받을 수 있습니다.

### Q2. 무료로 사용할 수 있나요?
네, 현재 베타 버전은 무료입니다. 무료 플랜은 최대 3개 단지까지 추적할 수 있습니다.

### Q3. 데이터는 어디서 가져오나요?
네이버 부동산의 공개 정보를 Tampermonkey 스크립트를 통해 수집합니다.

---

## 🚀 시작하기

### Q4. 어떻게 시작하나요?
1. 회원가입 (아이디, 이메일, 비밀번호)
2. Tampermonkey 스크립트 설치
3. 네이버 부동산에서 원하는 아파트 방문
4. "💾 저장" 버튼 클릭
5. Streamlit 앱에서 JSON 파일 업로드

자세한 내용은 `README.md` 참고

### Q5. Tampermonkey가 무엇인가요?
브라우저 확장 프로그램으로, 웹 페이지에 사용자 정의 스크립트를 실행할 수 있게 해줍니다. Chrome, Firefox, Edge 등에서 사용 가능합니다.

---

## 📊 데이터 수집

### Q6. 어떤 데이터를 수집하나요?
- 아파트명, 주소, 세대수
- 면적별 매매가/전세가
- 층수, 방향
- 투자금(매매가 - 전세가)

### Q7. 데이터가 실시간으로 업데이트되나요?
아니요. 사용자가 직접 네이버 부동산 페이지를 방문하여 "저장" 버튼을 눌러야 업데이트됩니다.

### Q8. 여러 아파트를 한 번에 저장할 수 있나요?
네! Tampermonkey에서:
1. 아파트1 방문 → "💾 저장"
2. 아파트2 방문 → "💾 저장"
3. "📥 전체 내보내기" → 하나의 JSON 파일로 다운로드

## ⚙️ 기능

### Q9. 투자금(갭)이란 무엇인가요?
매매가에서 전세가를 뺀 금액입니다. 갭투자 시 실제로 필요한 자기 자본 금액을 의미합니다.

**예시**: 매매가 10억, 전세가 8억 → 투자금 2억

### Q10. 관심 단지는 몇 개까지 등록할 수 있나요?
- **무료 플랜**: 최대 3개
- **프리미엄 플랜**: 무제한 (향후 출시 예정)

### Q11. 이메일 알림은 언제 오나요?
현재 베타 버전에서는 수동으로 테스트 이메일만 발송 가능합니다. 자동 알림 기능은 정식 출시 시 제공 예정입니다.

## 🔧 문제 해결

### Q12. 로그인이 안 돼요
1. 아이디/비밀번호를 확인하세요
2. 페이지를 새로고침해보세요
3. 쿠키가 차단되어 있지 않은지 확인하세요

### Q13. 데이터가 업로드되지 않아요
**확인 사항:**
1. JSON 파일 형식이 올바른지 확인
2. 파일 크기가 너무 크지 않은지 확인
3. Tampermonkey 스크립트가 최신 버전인지 확인

**해결 방법:**
1. 브라우저 콘솔(F12)에서 오류 메시지 확인
2. JSON 파일을 텍스트 에디터로 열어 형식 확인
3. GitHub Issues에 문의

### Q14. 새로고침하면 로그아웃돼요
쿠키가 활성화되어 있는지 확인하세요. 쿠키를 통해 30일간 로그인 상태가 유지됩니다.

### Q15. 자동 업로드가 작동하지 않아요
현재 기술적 제한으로 인해 자동 업로드는 완전히 지원되지 않습니다. 
대신 "📥 전체 내보내기" → JSON 파일 업로드 방식을 사용해주세요. (5초 소요)

## 💳 결제 & 플랜

### Q16. 유료 플랜은 언제 나오나요?
사용자 피드백을 수집한 후 2-3개월 내 출시 예정입니다.

### Q17. 유료 플랜 가격은 얼마인가요?
**예정 가격:**
- 베이직: ₩9,900/월 (20개 단지)
- 프로: ₩29,900/월 (무제한)

## 🔒 보안 & 개인정보

### Q18. 내 정보는 안전한가요?
네. 비밀번호는 bcrypt로 암호화되며, HTTPS를 통해 통신합니다. 
개인정보는 제3자에게 제공하지 않습니다.

### Q19. 회원 탈퇴는 어떻게 하나요?
현재 베타 버전에서는 이메일을 통해 요청해주세요. 
정식 버전에서는 설정 페이지에서 직접 탈퇴 가능합니다.
